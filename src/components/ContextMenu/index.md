## API

| 属性名    | 描述     | 类型            | 默认值 |
| --------- | -------- | --------------- | ------ |
| childern  | 子组件   | `ReactNode`     | --     |
| className | 类名     | `string`        | --     |
| style     | 行内样式 | `CSSProperties` | --     |

### ContextMenu.Item

| 属性名    | 描述         | 类型            | 默认值   |
| --------- | ------------ | --------------- | -------- |
| text      | 菜单文本     | `string`        | `(必选)` |
| onClick   | 点击菜单事件 | `() => void`    | `(必选)` |
| className | 类名         | `string`        | --       |
| style     | 行内样式     | `CSSProperties` | --       |
